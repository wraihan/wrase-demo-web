About me and this site


Overviews of the blog:-

  1. Built with Hugo (static website generator);
    • Chroma as the default syntax highlighter by Hugo;
    • Emoji function with emojify which is built in Hugo.
  2. Applies web framework (layouts) produced by Semantic UI;
    • Uses default icons shipped with Semantic UI framework (pretty much similar to Font Awesome);
    • Utilizes grid system which is also provided by Semantic UI.
